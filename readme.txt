== Description ==

This plugin gives you a small widget that actually retreives the lates lolcat and displays it along with a link to it. It caches the image for two hours so it won't delay.

**Usage**

You activate the plugin and then place the widget to your sidebar.

== Requirements ==

* GD library
* The cheezes folder within the lolcats_widget folder be writeable from the webserver.

== Installation ==

Copy lolcats_widget folder to your plugins directory and activate it on your plugins panel. Then add the widget to your panel. Make sure there is a "cheezes" folder within your plugin and it is writable by the webserver!
